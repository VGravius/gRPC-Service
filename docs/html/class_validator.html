Detailed Description

Класс для валидации входных данных и контроля уникальности request_id.

Предоставляет методы для проверки:

  • Имя параметра — не пустое.
  • Значение параметра — не пустое.
  • request_id — не пустой и уникальный.

Все методы, работающие с хранилищем used_ids_, потокобезопасны благодаря использованию std::mutex.

Note
Использует std::unordered_set для хранения использованных request_id.
Поиск и вставка выполняются за O(1) в среднем.

void Validator::MarkRequestIdUsed ( const std::string & request_id)

Отмечает request_id как использованный.

Метод потокобезопасен. После вызова этого метода IsRequestIdUsed() для данного ID вернёт true.

Parameters

Метод блокирует мьютекс и добавляет request_id в хранилище used_ids_. После этого любой вызов IsRequestIdUsed() для этого ID вернёт true.
