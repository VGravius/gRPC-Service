\doxysubsection{Detailed Description}
Потокобезопасное хранилище параметров в памяти. 

Класс \doxylink{class_cache}{Cache} предоставляет интерфейс для сохранения, получения и проверки наличия параметров в памяти. Все методы защищены мьютексом, что позволяет безопасно использовать объект из нескольких потоков одновременно.

\begin{DoxyNote}{Note}
Использует std::\+unordered\+\_\+map для хранения данных (O(1) доступ). 

std::\+mutex защищает данные от состояния гонки. 
\end{DoxyNote}
