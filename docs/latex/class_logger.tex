\doxysubsection{Detailed Description}
Класс для логирования операций в консоль. 

Предоставляет два основных метода:\+
\begin{DoxyItemize}
\item \doxylink{class_logger_aa81b667ce918f412a79dc613f34b7bde}{Log()} — для записи информационных сообщений об операциях.
\item \doxylink{class_logger_a58576dc8b35ac9cd61bd78baf873ed9e}{Log\+Error()} — для записи сообщений об ошибках.
\end{DoxyItemize}

Все сообщения автоматически дополняются временной меткой в формате "{}\+YYYY-\/\+MM-\/\+DD HH:\+\+MM:\+\+SS.\+mmm"{}.

\begin{DoxyNote}{Note}
Логи выводятся в стандартный поток вывода (std::\+cout). 

Время формируется в локальном часовом поясе. 
\end{DoxyNote}
