\doxysubsection{Detailed Description}
Класс для валидации входных данных и контроля уникальности request\+\_\+id. 

Предоставляет методы для проверки:\+
\begin{DoxyItemize}
\item Имя параметра — не пустое.
\item Значение параметра — не пустое.
\item request\+\_\+id — не пустой и уникальный.
\end{DoxyItemize}

Все методы, работающие с хранилищем used\+\_\+ids\+\_\+, потокобезопасны благодаря использованию std::\+mutex.

\begin{DoxyNote}{Note}
Использует std::\+unordered\+\_\+set для хранения использованных request\+\_\+id. 

Поиск и вставка выполняются за O(1) в среднем. 
\end{DoxyNote}
