\doxysubsection{Detailed Description}
Точка входа в g\+RPC-\/сервис управления параметрами. 

Этот файл содержит функцию \doxylink{main_8cpp_ae66f6b31b5ad750f1fe042a706a4e3d4}{main()}, которая запускает g\+RPC-\/сервер, и функцию \doxylink{main_8cpp_afec70f4aa13559a9e64d869bb52e2b2f}{Run\+Server()}, которая создаёт и настраивает сервер. Сервер слушает порт 50051 и обрабатывает запросы Get\+Parameter и Set\+Parameter.

Запускает g\+RPC-\/сервер и ожидает входящие запросы. 

Функция выполняет следующие шаги:\+
\begin{DoxyEnumerate}
\item Устанавливает адрес сервера (0.\+0.\+0.\+0:\+50051) — слушает все интерфейсы.
\item Включает рефлексию, чтобы утилита grpcurl могла автоматически определять методы сервиса.
\item Создаёт экземпляр класса \doxylink{class_telemetry_service_impl}{Telemetry\+Service\+Impl}, который содержит логику обработки запросов.
\item Настраивает Server\+Builder:\+ добавляет порт и регистрирует сервис.
\item Запускает сервер с помощью Build\+And\+Start().
\item Если сервер не запустился, выводит сообщение об ошибке и завершает работу.
\item При успешном запуске выводит сообщение в консоль и переходит в режим ожидания запросов (Wait()).
\end{DoxyEnumerate}

\begin{DoxyNote}{Note}
Сервер работает в бесконечном цикле, пока не будет прерван (Ctrl+C). 
\end{DoxyNote}
